# Snake Python Game 🐍😊
## Cara Memainkan

1. Buka terminal atau command prompt. 
   > git clone https://github.com/xcluster-crypto/snake-game.git
   
2. Navigasikan ke direktori tempat file `start.py` berada.
   > cd snake-game
   > pip install -r requirements.txt
   
3. Jalankan perintah berikut untuk memulai game:
   > python start.py
   
   ``buka melalui browser sesuai IP yang tertera...di terminal atau command prompt``
   
4. Setelah game dimulai, ikuti petunjuk berikut untuk memainkan game:
- Gunakan tombol panah (kiri, kanan, atas, bawah) untuk mengarahkan ular.
- Tekan tombol `Q` atau double tap pada layar untuk keluar dari game.
- Tekan tombol `P` untuk mereset game.


## Aturan Permainan

- Ular akan tumbuh lebih panjang setiap kali memakan makanan.
- Skor akan bertambah 10 poin setiap kali makanan dimakan.
- Kecepatan ular akan tetap konstan sepanjang permainan.
- Game akan berakhir jika kepala ular bertabrakan dengan tubuhnya sendiri atau jika jumlah permainan telah mencapai maksimum.
- Setelah game over, Anda dapat menekan tombol `P` untuk memulai kembali game.

## Informasi Tambahan

- Game ini mendukung kontrol sentuh pada perangkat layar sentuh.
- Tekan tombol `Q` atau lakukan double tap pada layar untuk keluar dari game dengan cepat.
- Game akan menampilkan skor tertinggi, skor saat ini, jumlah permainan yang tersisa, dan level saat ini di layar.

Selamat bermain dan semoga sukses!😊

Enjoy! 🐍😊
